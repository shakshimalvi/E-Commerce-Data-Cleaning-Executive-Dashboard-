# E-Commerce-Data-Cleaning-Executive-Dashboard-
📌 Project Overview

This project focuses on transforming a messy e-commerce dataset into a clean, structured, and executive-level interactive dashboard using Power BI and Power Query.

The objective was to apply data cleaning techniques, implement business logic, and create a single-page professional dashboard that provides meaningful business insights.

🎯 Project Objective

As a Business Intelligence Developer, the goals were to:

Clean and transform raw e-commerce data

Apply business logic transformations

Create calculated business metrics

Build a single-page executive dashboard

Generate data-driven business insights

🛠 Tools & Technologies Used

Power BI

Power Query

DAX (Data Analysis Expressions)

Data Cleaning & Transformation Techniques

Data Visualization & Dashboard Design

📈 Key Business Insights

The top-performing city contributes the highest revenue share.

Electronics category outperforms other categories in revenue generation.

A small group of top customers drives a significant portion of total revenue.

Moderate discounts positively influence revenue, but higher discounts do not always increase sales.

Sales show seasonal trends with noticeable monthly fluctuations.

🎨 Dashboard Design Standards Followed

Clean and professional layout

Consistent color theme (maximum 3 primary colors)

Proper alignment and spacing

Business-focused visual storytelling
